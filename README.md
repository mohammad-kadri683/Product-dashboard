# Product Dashboard

Frontend Developer Take-Home Assignment

## Live Demo
[View Live Demo](https://product-dashboard-beige-five.vercel.app)

## Features
- Product list view with pagination
- Real-time search with debouncing
- Category filtering
- Sorting options
- Product detail modal
- Shopping cart UI (localStorage)

## Tech Stack
- Next.js (App Router)
- TypeScript
- Bootstrap for styling
- React hooks
- SWR for API fetching

## How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`

## Notes
- The layout uses Bootstrap for styling.
- The page is not fully responsive.
